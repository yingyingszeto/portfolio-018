Color Names for CSS

We are lucky! 140 CSS color with names are supported by all modern browsers. When thinking about colors, I always prefer descriptive names than a string of dull letters or numbers.

In order to exposing myself to the variety of all the beautiful colors that have a name, I created this well-categorized color picking tools.

How will this tool help?

1. Copy to clipboard

Just simply click on the color button and the color name will automatically be copied. I found it extremely useful as some of the names are quite long.
