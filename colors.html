We are lucky! We have 140 CSS color with names that are supported by all modern browsers. When thinking about colors, I always prefer something descriptive than a string of letters or numbers.

In order to exposing myself to all the beautiful and unique CSS names, I created this well-categorized color picking tools.

How will this tool help?

1. Copy to clipboard

Just simply click on the color button and the color name will automatically be copied. I found it extremely useful as some of the names are quite long.

4. Light mode and Dark mode

Visibility of color on screen can be different between light and dark background. Checking the color with both backgrounds helps managing the visual attention of individual colors effectively.
