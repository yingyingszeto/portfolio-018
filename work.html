Sideline to Sideline - a contest was run by American headwear company, New Era Cap. Their customer receives a pin code upon purchase and signs up through the microsite to enter the campaign.

The microsite was created with WordPress with Gravity Form and plugins for generating the pin code. The big challenge of this project was ensuring the quality of the site and also testing and configuring the pin code redemption form system.

Things I have done:

1. Cross-platform and cross-device testing
2. Problem solving and troubleshooting
3. Adjusting default layout (CMS platform) to client's taste
4. Identifying potential technical problems and possible content improvements

Yuki Spa - a day spa in Markham. I designed and hand-coded for this project. The owner would like to expend the customer service to allow customers to communicate and make appointments through the website as it was hard for her to pick up every phone calls.

The goal is to increase online visibility and create the informative website with details of all the treatments, opening times, address etc. which bring convenient for both clients and the owner.

Things I have done:

1. Google API Map customed by Snazzy Maps with "Get Direction" button that allows customers to get the direction without having to type in the address manually.
2. Categorizing and organizing treatment menu with prices and details
3. Adding contract form that allows customers to make appointments or inquires
